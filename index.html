Creating Your First Website!

  1. First you will need to make an account on GitHub
  2. Once you have created your account, you will need to set up a repository.
        TIP Using a real name instead of an odd psuedonom can look more professional!
  3. Now that you have your GitHub ready to rock and roll, you need to download Visual Studio Code
  4. Once downloaded and installed you will then create a folder with both an index.html file and an image of yourself.
  5. Now head into Visual Studio and insert the following code: 
    
            html
                h1 Username's Portfolio /h1
                img src="./Image.JPEG" />
            /html
  6. Finally, go back into GitHub and upload both the index.html file and image file into your repository. BAM!
